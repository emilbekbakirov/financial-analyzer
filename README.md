# financial-analyzer
AI-powered financial analyzer that compares stocks using sentiment analysis, key ratios, and generates analyst reports

## What it does
Analyzes and compares multiple stocks using real-time news sentiment, 
key financial ratios, and AI-generated analyst reports.

## How to run
1. Install dependencies:
pip install yfinance transformers torch groq python-dotenv matplotlib numpy

2. Create a .env file with your Groq API key:
GROQ_API_KEY=your_key_here

3. Run:
python Financial Analyzer.py
